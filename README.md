# fastShopAdmin

##### 访问地址: [http://fastshop.applinzi.com/pmsAdmin#/login]( "fastShop"). 

## 说明

* 后台是基于Node.js的koa框架、mysql所构建的restfullApi

* 前端使用Vue全家桶、D2Admin后台模板、Element-UI组件构建的项目

* 后续会将该项目与仿网易严选H5重新整合在一起。目前只提供学习、参考的demo
